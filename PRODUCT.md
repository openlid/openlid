# Product

## Register

brand

## Users

OpenLid is for Mac users who need their laptop to keep working while the lid is closed: developers running long builds, coding-agent sessions, downloads, remote access, or desk workflows that should not be interrupted by sleep or screen lock. The primary site visitor is technical enough to understand menu bar utilities, Homebrew, CLI commands, GitHub releases, and privacy claims, but should not need to read architecture docs before deciding whether the app is safe to install.

## Product Purpose

OpenLid keeps a Mac awake with the lid open or closed while preserving local control, battery safeguards, and a native macOS feel. The website should make the promise clear, prove the utility is safe and local-first, and move visitors toward installing the signed macOS app or inspecting the open-source repository.

## Brand Personality

Technical, local-first, and restrained. The interface should feel like a dependable native utility rather than a SaaS control plane: quiet, exact, trustworthy, and comfortable around terminal workflows.

## Anti-references

Avoid generic SaaS launch-page patterns, decorative metric blocks, vague AI productivity language, cloud-dashboard aesthetics, heavy sales copy, and any design that makes OpenLid feel like it uploads, monitors, or intermediates user work. Avoid equal icon-card grids as the default explanation pattern.

## Design Principles

- Lead with the physical problem: a closed laptop should not kill active work.
- Prove local control before adding persuasion.
- Make install and source inspection obvious from the first viewport.
- Prefer technical specificity over marketing flourish.
- Keep the web surface isolated from the native app and GitHub Pages constraints.

## Accessibility & Inclusion

Target WCAG AA contrast, visible focus states, keyboard-operable navigation, responsive layouts down to small mobile widths, and reduced-motion-safe animation. Privacy and safety claims should be written plainly so cautious users can evaluate the app without guessing.
